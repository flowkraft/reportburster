# QuickStart — 5 minutes to a dashboard ⚡

Get a working dashboard quickly: either launch the built-in Portal or embed a sample report into a web page.

1. Start the Portal (Apps → WebPortal) and open *My Documents* to see the preloaded demo content. ✅
2. Or try embedding a component (copy/paste):

```html
<!-- Minimal embed: unified report component -->
<rb-report report-code="sales-summary" api-base-url="https://your-server/api/jobman/reporting" api-key="MY_KEY"></rb-report>
```

3. Use the parameter form to fetch data and explore table/chart/pivot views.

Tip: See **Embed** for component-specific examples and **Samples** for ready-to-run configs.
